Coffee review application


This app was inspired by people who have a hard time choosing a coffee brand from the grocery store. This web app will allow a user to see the review of a coffee brand so, when a user buys a coffee brand they will buy based on the reviews. That will give the user less stress to choose a coffee brand.
It is a full stack web application built in AngularJS on the frontend, with Express server running in Node.js, using Mongoose on the database design. It doesn't require signup to rate coffees, you can just add coffee brand rate the coffee.

Side Note: This app is currently in development mode. When rating it only accept image url for now.
